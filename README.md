# Tutorial: Portfólio do GitHub para Estudantes de TI

Guia prático de GitHub para iniciantes e desenvolvedores, com foco na criação de um portfólio profissional para estudantes de Tecnologia da Informação.

---

# 📚 Sobre o Projeto

Este repositório foi criado para ajudar estudantes de TI a aprenderem como organizar projetos no GitHub, criar READMEs profissionais e desenvolver um perfil atrativo para recrutadores e empresas da área de tecnologia.

---

# 🎯 Objetivos

- Aprender a usar o GitHub
- Criar um portfólio profissional
- Organizar projetos de forma correta
- Melhorar a apresentação dos repositórios
- Destacar habilidades técnicas
